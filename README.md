# Object Detection with YOLOv5
Веб‑приложение для детекции объектов на изображениях с использованием модели YOLOv5.

### Описание
Проект представляет собой веб‑сервис, который:
- принимает изображение от пользователя;
- анализирует его с помощью модели YOLOv5;
- отображает изображение с выделенными объектами и их названиями.

### Функциональные возможности

- Загрузка изображения через веб‑интерфейс.
- Детекция и отображение объектов на изображении.
- Виден список обнаруженных объектов с вероятностями.

1. Установить необходимые зависимости: pip install -r requirements.txt
2. Запустить приложение командой: python app.py

*ПРИМЕЧАНИЕ: при наличии интернета модель автоматически установится, в ином случает перед запуском приложения скачайте модель по ссылке и поместите её в корень проекта.*
